App is Building

  1. Use any One of the Following tab to create your App
  2. Use the Editor to create your own template.
  3. Use Embed Website tab to convert your website into an android app.
  4. In embeded website put your own website url and title instead of 'YOUR WEBSITE URL' and 'YOUR WEBSITE TITLE'.
  5. Use http or https along with your website url eg: https://droidota.tech
  6. In Paste tab paste your HTML code if you have it already.
  7. Once done hit the Render button to render it to phone
  8. If the edited HTML suits you then confirm the build by Build button
  9. You are allowed to build max 2 apps with one gmail id
  10. If you want more features or build write to us Mail Us
